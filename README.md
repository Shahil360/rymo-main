# E-Commerce Web Application

A full-stack e-commerce web application built using **FastAPI (Python)** and **MySQL**. Users can browse products, view details, add items to a cart, register/login, and place orders. Admins can manage products through the backend.

## Features
- Browse products by category
- Product detail pages
- Shopping cart and checkout flow
- User registration & login
- Order creation
- Admin product management (add / edit / delete)
- Responsive frontend (HTML / CSS / JavaScript)

## Tech Stack
### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python
- FastAPI
- MySQL
- Uvicorn (development server)

## Getting Started

### Prerequisites
- Python 3.8+
- MySQL server
- pip

## Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/Shahil360/E-Commerce_Web_Application
cd E-Commerce_Web_Application
pip install -r requirements.txt
```
### 2. Install dependecnies
```bash
pip install -r requirements.txt
```

### 3. Run the backend
```bash
cd backend
uvicorn main:app --reload
```
